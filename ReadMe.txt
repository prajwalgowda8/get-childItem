
----------------------------------------------------------------------------------------------How to use------------------------------------------------------------------------

** Download the Scripts.zip folder and extract it.



----------------------------------------------------------------------------------------------Step 1------------------------------------------------------------------------

Running for Single drive:
---------------------------------

Running Get-FolderAndFileList.ps1 script:
-------------------------------------------------------------------

1.Please open "Get-FolderAndFileList" Folder.
2. Please open the "Get-FolderAndFileList.bat" file.
3.Modify the drive name in the .bat file (eg:C/D)
4.Save the .bat file
5.Right click on it and run the .bat file with admin rights.

Note: 1. If there is multiple drives, Kindly create muliple .bat file with diffrent Drive name.
           


----------------------------------------------------------------------------------------------Step 2-------------------------------------------------------------------------

Running Get-ConsolidatedReport.ps1 script:
-----------------------------------------------------------

1.Please open "Get-ConsolidatedReport" Folder.
2. Edit the run.bat and change the path.
3. Provide path with double slashes (eg: C:\\users\\Downloads).
4. Run the run.bat as admin.
5. Two files Accessible.csv and NotAccessible.csv will be created.
